# SNL Website — SEO/GEO Playground

SuperNova Leap public website. Built as an SEO and GEO (Generative Engine Optimization) testing ground.

## Stack
- **Hosting:** Netlify
- **Domain:** supernovaleap.com
- **Deploy:** Git push → Netlify auto-deploy

## Getting Started
```bash
# Local dev
npx serve .
# or
python -m http.server 8080
```

## Deploy
Push to `main` → Netlify auto-deploys.
